# 🚗 Car Rental Management System

A robust desktop application built using **Java Swing** and **AWT**. This project was developed as a part of my **Summer Training** to digitize the vehicle rental process, focusing on user experience and efficient data management.

## 📋 Project Overview
The Car Rental Service application provides a seamless flow for customers to register, browse available vehicles, calculate rental costs, and complete payment processes. It uses a modular design with a single-frame multi-panel interface.

## ✨ Key Features
* **Multi-Page Interface:** Implemented using `CardLayout` for smooth transitions between Welcome, Registration, Selection, and Billing screens.
* **Customer Registration:** Securely collects user information including name, contact details, and address.
* **Dynamic Fleet Management:** Displays car details, IDs, and pricing per day.
* **Automated Fare Calculation:** Real-time calculation of the total bill based on the duration of the rental.
* **Payment Integration UI:** Support for various payment modes including UPI and Cash.

## 🛠 Tech Stack
* **Language:** Java (JDK 8 or higher)
* **GUI Framework:** Java Swing & AWT
* **IDE:** Visual Studio Code
* **Version Control:** Git

## 📂 Project Structure
* `CarRentalService.java`: The main entry point containing the logic for navigation and UI components.
* `*.class`: Compiled bytecode files (Note: Ideally excluded via .gitignore).
